## About project

This application allows users to create ToDoLists, edit them and share with other users

### Instaliation

- clone project
-
- ```shell
  make install
    ```
- open http://to-do-list.local:81 or http://localhost:DOCKER_HTTP_PORT


### Checklist

- [x] Регистрация / авторизация пользователей для создания личных списков. Возможность редактирования сохраненных списков

- [x] Поиск по элементам списка. Фильтрация элементов списка по тегам (одному или нескольким)
- [x] Возможность расшарить список другому пользователю (т.е. пользователь А может дать доступ на чтение пользователю Б)
- [x] Разграничение прав доступа к списку (пользователь А может только читать, пользователь Б может читать и редактировать)
